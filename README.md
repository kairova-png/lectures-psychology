# Психология дәрістері

Екі тақырыпты қамтитын интерактивті презентация

## 🎓 Тақырыптар

### 1️⃣ Өмірдің мәні және кәсіби өзін-өзі анықтау психологиясы
- Өмірдің мәні, өмірлік мақсаттар және бағдарламалар
- Кәсіби маманның сенімдері мен бағдарлары
- Қазақстанның заманауи даму кезеңіндегі кәсіби тұлғаның құндылықтары
- Кәсіби өзін-өзі анықтаудағы жеке құндылықтар мен мағыналар
- Кәсіби өсудегі кедергілер, эмоциялық қажу синдромы

### 2️⃣ Тұлғаның денсаулық психологиясы
- Психика мен дененің өзара байланысы және өзара ықпалы
- Аурулардың пайда болуы мен денсаулықты нығайтудың психологиялық факторлары
- Психикалық белсенді заттар мен зиянды әдеттер
- Салауатты өмір салты — жетістікке жету негізі
